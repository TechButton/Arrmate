H.265 Transcoder

Convert your media library to H.265/HEVC to reduce file sizes by 30–50%. Use the Command page to start a transcode job with natural language.

How to start a transcode job

Go to the Command page and type something like:

convert all my movies to H265
transcode Breaking Bad to H265 to save space
convert all TV shows to HEVC

Requires ffmpeg installed and your media directories mounted into the Arrmate container at the same paths that Sonarr/Radarr report. See docker-compose.yml for volume examples.
