Quick-start for Android / API clients

1. Get a token (or create one above):

POST /api/v1/auth/token
Content-Type: application/json

{
  "username": "your-username",
  "password": "your-password",
  "token_name": "Android App"
}

2. Use the token in every request:

POST /api/v1/execute
Authorization: Bearer amt_your-token-here
Content-Type: application/json

{
  "command": "add Interstellar to Radarr"
}

3. Other useful endpoints:

GET  /api/v1/user           → current user info
GET  /api/v1/services       → configured services & status
GET  /api/v1/config         → app configuration
DELETE /api/v1/auth/token   → revoke current token (logout)
